## Development

### Clone the repository

$ `git clone  https://git.thelanding.page/tlp/tag.git`

### Navigate into the directory

$ `cd tag`

### Build the website

$ `make`

### Run the website

$ `./website`

### View the website

[http://localhost:4507](http://localhost:4507)
